# 70. Climbing stars

- 문제링크: https://leetcode.com/problems/climbing-stairs/description/
- 난이도: 미디움

# 풀어보기


```python
class Solution:
    def climbStairs(self, n: int) -> int:
        dp = [0 for i in range(n)]
        if n <= 2:
            return n
        dp[0] = 1
        dp[1] = 2
        for i in range(2, n):
            dp[i] = dp[i-2] + dp[i-1]
        return dp[-1]
```
