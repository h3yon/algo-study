# 70. Climbing stars

- 문제링크: https://leetcode.com/problems/climbing-stairs/description/
- 난이도: 미디움

# 풀어보기


```python
public int climbStairs(int n) {
    if (n <= 1) {
        return 1;
    }

    int[] dp = new int[n + 1];
    dp[1] = 1;
    dp[2] = 2;

    for (int i = 3; i <= n; i++) {
        dp[i] = dp[i - 1] + dp[i - 2];
    }

    return dp[n];
}
```
